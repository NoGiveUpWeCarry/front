## 프로젝트 소개
> **PAD**는 개발자, 디자이너, 아티스트가 한 공간에서 프로젝트를 만들고 팀원을 매칭하여 실시간으로 협업할 수 있는 **소셜 커뮤니티** 서비스입니다.

<br />

## 서비스 기능
- 피드 작성 기능
- 프로젝트 모집, 지원 기능
- 1:1 채팅, 그룹 채팅 기능
- 알림 기능
- 마이페이지, 계정 설정 기능
- 검색 기능

<br />

## 설치
```
$ git clone https://github.com/NoGiveUpWeCarry/front.git
$ pnpm install
$ pnpm run dev
```
